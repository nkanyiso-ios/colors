
# colors
A basic app that can convert RGB colour values to hexadecimal colour values.

How to run the application:\
	1) Download project\
	2) Unzip Folder\
	3) Open terminal and navigate to the folder in step 3\
	4) Run pod install\
	5) Run the app on simulator
